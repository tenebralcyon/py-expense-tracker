\# py-expense-tracker



A simple command-line expense tracker written in Python.



\## Features

\- Store expenses in a local SQLite database

\- Add, list, delete expenses

\- Import expenses from a CSV file

\- Categorize automatically using keyword rules (priority-based)

\- Set monthly budgets by category

\- Show totals (overall or by month)

\- Export a month to CSV



\## Requirements

\- Python 3.x
